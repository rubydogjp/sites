# Rubydog Sites

Rubydog JP に関する各種 Web サイトを管理するリポジトリ

## 開発

git で `main` ブランチで開発し、`apex`, `gitnote` など対応するブランチへ push すると GitHub Actions によって対応するフォルダがデプロイされる.


| ブランチ | 公開サイトURL |
|---|---|
| apex | [rubydog.jp](https://rubydog.jp) |
| gitnote | [gitnote.rubydog.jp](https://gitnote.rubydog.jp) |
| packages | [packages.rubydog.jp](https://packages.rubydog.jp) |
