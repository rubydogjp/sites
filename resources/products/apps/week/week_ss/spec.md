# Week PR画像仕様書

## 概要

Week（時間割）アプリのアナウンス・紹介サイト用PR画像。

## SS画像 (スクリーンショット) — 8枚

アプリの主要画面をモバイル/デスクトップの両方でキャプチャした透過PNG。

### 出力仕様

| 項目 | 値 |
|------|-----|
| フォーマット | PNG (透過背景) |
| 角丸 | モバイル: 24px / デスクトップ: 8px |
| 影 | なし |
| pixelRatio | 2.0 |

### モバイル (PhoneFrame)

| 項目 | 値 |
|------|-----|
| コンテンツサイズ | 390×844 |
| 出力サイズ | 780×1688 (@2x) |
| ステータスバー | あり (9:41, Wi-Fi, バッテリー) |
| 角丸 | 24px |

### デスクトップ (BrowserFrame)

| 項目 | 値 |
|------|-----|
| コンテンツサイズ | 1200×750 |
| 出力サイズ | 2400×1500 (@2x) |
| タイトルバー | macOS風 (信号機ドット + URLバー `https://www`) |
| 角丸 | 8px |
